class nyawc.Queue.Queue(options)[source]¶

Bases: object

A ‘hash’ queue containing all the requests of the crawler.

Note: This queue uses a certain hash (from __get_hash) to prevent duplicate entries and improve the time complexity by checking if the hash exists instead of iterating over all items.
